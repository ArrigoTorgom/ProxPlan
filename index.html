Case Columns

These column labels apply wherever a table shows values across design cases (Project Parameters, Process Configuration blocks). By default there is a single "Value" column. If this tender needs a range of operating cases, add more columns and rename them — e.g. Min / Ave / Max — on an as-required basis.

Templates

Start from a ready-made flowsheet shape. Loading a template replaces everything currently in the editor — save a draft first if you want to keep what's there.

Save & Load

Everything lives in this JSON file — there's no separate server or account. Save Draft keeps your data, Save as Template strips it back to a blank starting point, and Import loads either kind of file back in.
